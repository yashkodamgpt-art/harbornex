# HarborFlow VS Code Extension

Deploy your projects to your own hardware with one click.

## Features

- **Deploy to Harbor** - Right-click any folder to deploy
- **Status Bar** - Shows connection status
- **API Key Config** - Store your API key securely

## Commands

- `HarborFlow: Deploy to Harbor` - Deploy current project
- `HarborFlow: Set API Key` - Configure your API key
- `HarborFlow: Show Status` - Check connection status

## Setup

1. Copy this folder to your VS Code extensions directory
2. Restart VS Code
3. Run `HarborFlow: Set API Key` from Command Palette
4. Right-click a folder and select "Deploy to Harbor"
